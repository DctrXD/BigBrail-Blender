## 📦 Installation

1. Clone or download this repo.  
2. From project root:
   ```bash
   zip -r bigbrain.zip bigbrain
   ```
3. In Blender:  
   `Edit > Preferences > Add-ons > Install...` → selecione `bigbrain.zip`  
4. Ative **BigBrain** e configure em Preferences → Add-ons.
